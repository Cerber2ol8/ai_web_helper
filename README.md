# AI Web Helper 项目说明

## 项目简介
本项目是一个基于Web的AI辅助工具，提供代码智能补全、文档生成等功能。

## 安装指南
1. 克隆仓库：`git clone https://github.com/yourname/ai_web_helper.git`
2. 安装依赖：`npm install`

## 使用说明
- 启动开发服务器：`npm start`
- 构建生产环境：`npm run build`

## 功能特性
- 实时代码补全
- 文档智能生成
- 代码质量分析
- 交互式编程助手
