功能简介

插件功能：一键将网页内容发送给 AI，帮你快速生成摘要（Chrome 浏览器）。

安装方式

  1. 下载插件包并解压；
  2. 打开 chrome://extensions；
  3. 右上角开启开发者模式；
  4. 点击“加载已解压的扩展程序”，选择该插件目录；
  5. 启用插件。

基本使用

在插件选项页填写你在该应用中的 服务端链接（HTTP/HTTPS）。之后在任意页面点击浏览器工具栏按钮，或页面右侧浮动按钮，选择 Send → AI 即可。

① 获取服务端链接
② 一键发送

点击 Send → AI。扩展会抓取可见文本（可选截图，暂未实装）并发送到你的服务端。

③ 快速总结

你的服务端调用 LLM 生成摘要（可返回 Markdown），窗口内实时流式展示。

④ 安全与可控

所有数据经你配置的服务端转发与处理，便于审计与限流。

常见问题

Q：为什么看不到按钮？
A：确保已启用扩展，并在选项中开启“页面悬浮按钮”。

Q：提取不到文字？
A：某些站点采用反爬/虚拟化渲染。“提取模式”为 元素 + innerText，请尝试滚动页面后再发送。

Q：请求失败？
A：检查服务端地址是否可访问（协议、端口、CORS），以及 API Key 配置。

隐私说明

网页内容只会发送到你配置的服务端；不经过第三方。你可以在服务端实现脱敏、日志与配额控制。

开发者友好

提供可配置的请求头/负载结构，便于与任意后端（如 Flask、FastAPI、Node）对接。
